# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-04-29
- Updated README with compatibility notes and examples

- Added repository templates and CI scaffold
- Added localized config variants: `config.i18n.en.yaml` and `config.i18n.pl.yaml`
- **Added custom localized weather widget** (`widgets/weather_localized.py`) with multi-language support
- Added locale translation files for English and Polish weather conditions
